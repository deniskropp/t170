# Raw Task Action Statements (TAS) Extracted from Playbook
## Operational Cycle Management
Identify and prioritize tasks from the plan.
Update task states (pending, in_progress, completed, cancelled).
Assign tasks to personas and facilitate execution.
Enforce dependencies before task initiation.
Escalate tasks requiring external info or decisions.
Monitor task progress and system performance.
## Meta-Communication
Provide process updates and summaries.
Explain structural decisions.
Clarify requirements.
Initiate discussions on ambiguities.
## Ethical Guidelines
Identify and mitigate biases.
Review content for ethical compliance.
Facilitate ethical discussions.
## Natural Language to Formal Translation
Identify key concepts in NL.
Map NL elements to formal elements.
Derive formal structure.
Validate formal structure.
## Dynamic Role Adaptation
Identify need for role adaptation.
Propose role change.
Review and approve adaptation.
Update role definitions.
Refine system prompts.
Communicate changes.
## Joint Decision-Making
Identify decision point.
Define scope and stakeholders.
Gather information.
Discuss and deliberate.
Formulate recommendation/decision.
Document decision.
Communicate decision.
## Integrity and Security
Verify sources.
Validate data inputs.
Control changes.
Audit data modifications.
Scan for vulnerabilities.
Respond to incidents.
## Plan Generation
Gather input for plan.
Define objectives.
Break down tasks.
Allocate resources.
Establish timeline.
Assess risks.
Review and refine plan.

Define complex problems.
Synthesize information.
Ideate solutions.
